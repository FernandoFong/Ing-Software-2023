Después de las instalaciones de MySQL y que hayamos podido conectarnos ya sea
vía línea de comandos o vía workbench verificar que se tenga instalado:

PyMySQL

En caso de no tenerlo, entonces simplemente se debe de ejecutar el comando: 

pip install PyMySQL --user

Esto instalará la biblioteca de PyMySQL y estará disponible para todo el sistema.

Posteriormente, se puede realizar la ejecución del programa como sigue:

python main.py
